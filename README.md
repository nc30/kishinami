# 岸波 Kishinami

for Raspberry Pi notifaction module software.
communicate on AWS IoT. receive signal and state.

## Install

```shell
$ git clone https://github.com/nc30/kishinami.git
$ cd kishinami
$ ./install.sh
$ sudo systemctl enable kishinami
$ sudo systemctl start kishinami
```
